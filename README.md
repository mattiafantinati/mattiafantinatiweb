# Sito HTML Statico

Questo è un sito HTML statico creato per una persona. Include le seguenti sezioni:

- **Biografia**: Una breve descrizione della persona.
- **Portfolio**: Una lista dei progetti o lavori principali.
- **Contatti**: Informazioni di contatto.

## Struttura del Progetto

- `index.html`: La homepage del sito.
- `styles.css`: Foglio di stile per il design del sito.

## Come Visualizzare il Sito

1. Clona il repository o scarica i file.
2. Apri `index.html` in un browser web.

## Personalizzazione

Modifica i contenuti di `index.html` per aggiungere informazioni personalizzate.
